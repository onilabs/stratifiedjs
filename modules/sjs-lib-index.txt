@lib     StratifiedJS Standard Module Library
@version 1.0.0
@summary Standard collection of modules for the StratifiedJS language
@desc
    This Library is the standard collection of modules
    accompanying the [StratifiedJS](http://onilabs.com/stratifiedjs) language.


    ### Loading `sjs` Modules

    You can load modules from the `sjs` library by using the builtin `sjs:` scheme, e.g.:

        var http = require('sjs:http');
    
    See [#language/builtins::require] for more information about the module system.
